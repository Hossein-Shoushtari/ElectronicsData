
These data is one of the first real data sets with 5G simulation concept. This is measured in HCU-building (Henning-Voscherau-Platz 1, 20457 Hamburg) on forth floor.
Start and End-Position is on the same point. The number of step is 220.

The test device is: Samsung S10 5G (built in 2020)

This dataset can be used for approaches with PDR, PF and KF.

You can read the data by: reader.ipynb

the paper is published: https://www.mdpi.com/2079-9292/10/4/397/htm
